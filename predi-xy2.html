      • Predi-XY

Predi-XY is a Python-based, homemade utility for generating predicted NICS-XY-Scans of chain-like polycyclic aromatic hydrocarbons, using a novel additivity scheme: Piecing it Together: An Additivity Scheme for Aromaticity using NICS-XY-Scans.

At the moment, the program is limited to acenes (tetracenes, pentacenes, etc.) of any length and topology, and can be used only on Windows.
An expanded version, capable of treating also heteroatom-containing systems, will be released in the future.

To use Predi-XY, click on the download link below. The program is a single-file executable (due to this, it is a bit slow at startup - please allow a few seconds). Please report any bugs you may find.

If you use Predi-XY, please cite the following:

 1.  R. Gershoni-Poranne, A. Stanger,  Chem. - Eur. J.  2014, 20, 5673-5688.
