NICS-Visualizer

NICS-Visualizer is a local web application for the interactive visualization of NICS-XY-Scan results computed with Gaussian. It reads .armdat files produced by the Aroma package and displays per-ring NICS profiles, an interactive 2D structure panel showing which ring corresponds to which scan center, and a 3D molecular viewer with ghost-atom positions highlighted.

Main features:

  • Interactive per-ring NICS-XY-Scan plots (NICSzz, NICSπzz, 3Δiso)
  • Clickable 2D molecular structure — each ring linked to its scan center
  • View the Gaussian log file for any individual scan center
  • 3D molecular viewer with ghost atom (Bq) positions and sigma-center tabs
  • Reads center log files directly from zip archives

NICS-Plotter

NICS-Plotter is a local web application for plotting and comparing NICS-XY-Scan results across multiple molecules and ring centers. It reads .armdat files produced by Aroma and provides an interactive, publication-ready plotting interface.

Main features:

  • Load entire directories of .armdat files at once
  • Select molecules and ring centers to overlay on a single plot
  • Customize colors, line styles, font, and plot dimensions
  • Export publication-quality SVG/PNG figures
  • Interactive 2D molecular structure panel with clickable rings
  • 3D molecular viewer and log file viewer per ring center
